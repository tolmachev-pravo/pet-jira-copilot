Предложенные ворклоги теперь можно настраивать перед добавлением, а логика их привязки стала гибче.

## Настройка перед добавлением

Для каждой записи в предложенном ворклоге в меню добавлено действие **«Добавить произвольный ворклог»**. По клику открывается привычное окно, преднастроенное текущими параметрами записи — можно изменить время и комментарий.

## Улучшения интерфейса

Кнопки и иконки справа от каждой записи приведены к общему стилю приложения: прямоугольная форма вместо скруглённой. Кнопка **«Добавить»** осталась кнопкой, а отметка для уже добавленных записей теперь отображается иконкой-галочкой.

## Гибкая привязка ворклогов

Раньше жёсткая связь между фактическим временем начала и предложенным временем окончания создавала ограничения. Теперь привязка рассчитывается по максимальному совпадению диапазона времени выполнения — можно свободно комбинировать ворклоги из разных источников, например добавленные через интерфейс Jira.
