Теперь можно создать произвольный ворклог — например, списать время на задачу, которой нет среди предложенных.

## Что это даёт

- Списание времени на любую задачу, а не только на предложенные.
- Быстрый старт: дата и время предзаполнены началом рабочего дня.
- Свободный комментарий к ворклогу.

## Как пользоваться

1. В строке дня откройте контекстное меню (**⁝**).
2. Выберите **«Добавить произвольный ворклог»** — откроется окно с преднастроенными датой и временем.
3. Выберите задачу, при необходимости укажите комментарий и подтвердите.

> 💡 Список задач по умолчанию соответствует закэшированным элементам (параметр `CachedIssues` в конфигурации). Если нужной задачи нет — введите её номер в поиске, и приложение подтянет данные из Jira. Если комментарий не указан, он записывается в формате «Discussion [номер задачи]».
